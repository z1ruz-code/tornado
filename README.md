## Installation and Run

1. **Clone the repository**
```bash
git clone https://github.com/z1ruz-code/tornado.git
cd tornado
```
2. **Install dependencies**

Make sure you have [Python 3.6](https://www.python.org/) or higher installed, then run:
```bash
pip install -r requirements.txt
```
3. **Run the program**
```bash
python tornado.py
```
## Description of the Tool

**Tornado** is a command-line tool for encrypting and decrypting messages with multilingual support.

When you encrypt text, you receive a key (it's best to save it in a secure place right away) and a token. Without each other, they won't be able to decrypt the text.

## Language of tool

Using the `translations.json` file, you can add any language to the tool.
